\chapter*{Rezumat}

Lucrarea analizeaza structura relatiilor dintre sectoare ale pietei financiare utilizand metode statistice si metode de analiza a retelelor. Analiza se bazeaza pe randamente zilnice ale unor ETF uri sectoriale obtinute din baza de date Yahoo Finance.

Dependentele dinamice dintre variabile sunt modelate folosind un model vector autoregresiv. Reziduurile modelului sunt utilizate pentru estimarea matricii de covarianta.

Pentru identificarea dependentei conditionale dintre active este utilizata metoda graphical lasso care estimeaza matricea de precizie. Structura acestei matrici este interpretata ca o retea financiara in care nodurile reprezinta sectoare iar muchiile reprezinta relatii conditionale intre acestea.

Analiza retelei permite identificarea sectoarelor centrale si a structurii de interconectare a pietei financiare.
